Sprite converter for NDS
===

Convert sprites to NDS matrix


![alt text](https://raw.githubusercontent.com/Asiern/SpriteNDS/master/SpriteNDS.png)

---

#### Supported files
| File extension | Support | 16x16 | 32x32 |
|----------------|---------|-------|-------|
| .jpg           | Yes     | Yes   | Yes   |
| .png           | Yes     | Yes   | Yes   |
| .bmp           | Yes     | Yes   | Yes   |
| .gif           | Yes     |   ?   |   ?   |
| .mpo           | No      |   -   |   -   |
| .jpf           | No      |   -   |   -   |
| .iff           | No      |   -   |   -   |
| .dcm           | No      |   -   |   -   |
| .psd           | No      |   -   |   -   |
| .jps           | No      |   -   |   -   |
